合并两个不同得项目，就是原本一个项目开发到一半打算重构，但是重构时完全新建仓库了，现在新得仓库提交了很多，如何把新仓库提交合并到老的仓库


```sh
# 1. 进入老仓库目录
cd /path/to/old/repo

# 可先对老项目当前状态新建分支
git branch old

# 这时可以完全移除所有文件并提交
git commit -m "remove old"

# 2. 添加新仓库作为远程仓库（如果还没添加）
git remote add new-repo /path/to/new/repo 

# 3. 拉取新仓库代码
git fetch new-repo

# 4. 强制合并不同历史的仓库
git merge new-repo/master --allow-unrelated-histories

# 5. 处理冲突（如果有）
# 编辑冲突文件，然后执行：
git add .
git commit

# 6. 推送结果
git push origin master
git push origin old
```
